The Front End Drupal Summit

A quasi-official Drupalcon event

New Orleans Convention Center - Room 275

9am - 5pm, May 9, 2016

Join fellow front end developers, themers, designers, UX practitioners and more for a free day-long working session at the New Orleans Convention center, Monday before Drupalcon.

We'll discuss, debate, share, sketch and hopefully achieve consensus as we dive into the following topics:

  • Working with Twig and D8 Theming
  • Building decoupled JS-powered front ends
  • Evolving Drupal's component-based future
  • Improving Drupal's admin UX
  • Other stuff as may come up

The front end landscape has never been so wide open. If you've ever wanted to know how Drupal is made, or wanted to help shape its direction, you should come to this summit.

This event adheres to both the Drupal Code of Conduct as well as the event-specific Drupalcon Code of Conduct.
